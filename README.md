# Personal Website - Abdulrazzak Swai

Welcome to my **personal website!** This site is my digital portfolio, highlighting my **skills, certifications, projects, ways to connect, and more**.

Here, you can explore:

- **About Me:** An interactive terminal-style component detailing my background and journey
- **Highlights:** Key milestones and recognitions in my career and education
- **Statistics:** A visual breakdown of my learning progress and achievements
- **Projects:** Technical projects and contributions highlighting a range of skills
- **Credentials:** A dynamic tabbed interface displaying my completed Certifications and Courses
- **Momentum:** Track my current focus, ongoing tasks, and recent goals
- **Get In Touch:** Connect with me via LinkedIn, GitHub, Hack The Box, and TryHackMe

## ✨ Key Features

- Dynamic content generation using JavaScript and JSON (for highlighting about me, projects, credentials, etc.)
- Light & Dark theme toggle for accessible viewing
- Interactive cursor elements and scroll progress indicator
- Modern and responsive layout leveraging a custom glassmorphism aesthetic
- Links to my cybersecurity writeups and technical walkthroughs

## 🛠️ Technologies Used

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome
- Custom Web Fonts
